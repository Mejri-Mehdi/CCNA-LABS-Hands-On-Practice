# CCNA-LABS-Hands-On-Practice

## Hands-on CCNA Labs – Switching, Routing, Security & Troubleshooting

This repository contains my comprehensive hands-on lab practice for the **Cisco CCNA** certification, completed using **Cisco Packet Tracer**. Over the course of this journey, I have built, configured, and troubleshooted a wide range of networking scenarios covering fundamental to advanced topics.

---

## About This Repository

This repo serves as a documentation of my practical experience and understanding of core networking concepts. Each lab was designed to reinforce real-world skills required for the CCNA exam and beyond. All configurations were performed from scratch on Cisco devices (routers, switches, firewalls, wireless controllers, and endpoints).

---

## Topics Covered

### 🔹 Switching & VLANs
- VLAN creation and assignment
- Trunking (802.1Q)
- VTP (VLAN Trunking Protocol)
- STP (Spanning Tree Protocol) – root bridge election and port roles
- EtherChannel (LACP, PAgP, Static)
- Inter-VLAN routing (legacy, router-on-a-stick, SVIs)

### 🔹 Routing Protocols
- **Static Routing** – IPv4 & IPv6, floating static routes
- **OSPF** – single-area, multi-area, passive interfaces, authentication, route redistribution
- **EIGRP** – unequal-cost load balancing, variance, passive interfaces
- **Route Redundancy** – HSRP, FHRP

### 🔹 Network Services
- DHCP (server, client, relay agent)
- DNS
- NTP (Network Time Protocol)
- SNMP (Simple Network Management Protocol)

### 🔹 Security
- Device hardening (passwords, SSH, enable secret)
- Standard and Extended ACLs (numbered and named)
- NAT (Static, Dynamic, PAT)
- DHCP Snooping
- Dynamic ARP Inspection (DAI)
- Wireless Security (WPA2+PSK on WLC)

### 🔹 WAN Technologies
- GRE Tunnels
- Serial connections
- PPP and HDLC (implied)

### 🔹 IPv6
- Addressing (SLAAC, EUI-64)
- IPv6 static routing
- Dual-stack configuration

### 🔹 Network Management & Troubleshooting
- CDP (Cisco Discovery Protocol)
- LLDP (Link Layer Discovery Protocol)
- Simulation mode analysis
- Packet capture and traffic analysis
- IOS upgrades (TFTP/FTP)
- System logging and NTP

### 🔹 Wireless Networking
- WLC (Wireless LAN Controller) configuration
- Lightweight APs (LAPs)
- WLAN creation (Internal/Guest)
- WPA2+PSK security
